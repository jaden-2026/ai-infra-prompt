说明：本配色方案适用软件平台设计，包括前端和后端web页面。
# 全局规范
### 配色规范
**主色**
+ Primary: Indigo（靛蓝色 #4B55D9）
+ 用途：按钮、导航栏、重点高亮、主操作入口
+ 2. 辅助色
+ Success：#00B42A（成功 / 完成）
+ Warning：#FF7D00（警告 / 提醒）
+ Danger：#F53F3F（错误 / 删除 / 危险操作）
+ Info：#86909C（信息 / 次要文案）

**中性色**
+ 标题文字：#1D2129
+ 正文文字：#4E5969
+ 次要文字：#86909C
+ 分割线 / 边框：#CDD0D6
+ 背景底色：#F2F3F5
+ 纯白背景：#FFFFFF

**样式**
+ 最小间距：4px
+ 小间距：8px
+ 常规间距：16px
+ 大间距：24px
+ 超大间距：32px
+ 模块边距：40px

**圆角**
+ 无圆角：0px
+ 小型圆角：4px（按钮、输入框）
+ 中型圆角：8px（卡片、弹窗）
+ 大型圆角：16px（头像、特殊模块）

**全局状态设计**
+ 状态管理使用 Redux
+ Empty   : none 
+ Loading :全局 / 局部骨架屏、加载动画
+ Error   :错误提示、重试按钮、兜底展示

### 全局组件规范
**按钮**
+ 主按钮：靛蓝背景 + 白色文字
+ 次按钮：白色背景 + 靛蓝边框
+ 文字按钮：无背景无边框，仅文字
+ 禁用状态：透明度 0.5，不可点击

**输入框**
+ 默认：灰色边框 + 白色背景
+ 聚焦：靛蓝边框 + 阴影
+ 错误：红色边框 + 错误提示文案

**弹窗**
+ 标题居中 / 左对齐，底部固定操作按钮
+ 确认按钮：主色，取消按钮：次要文字色

### 全局交互规范
+ 点击反馈：所有可点击元素添加 hover/active 状态
+ 表单提交：提交后按钮置为加载状态，禁止重复提交
+ 异常处理：接口报错统一提示，无数据展示空状态页面
+ 路由跳转：页面切换无刷新，滚动位置合理复位
### 全局动画规范
+ 过渡时长：0.3s
+ 过渡曲线：ease-in-out
+ 适用场景：弹窗显隐、元素切换、hover 效果
### 全局字体规范
+ 字体族 PingFang SC, Microsoft YaHei, sans-serif
+ 字体层级
+ 超大标题：24px / 700
+ 页面标题：20px / 600
+ 模块标题：18px / 600
+ 正文内容：16px / 400
+ 次要文案：14px / 400
+ 辅助说明：12px / 400

## 图表规范
### 饼图规范
**布局与尺寸**
+ 占比：容器内居中展示，饼图直径占容器有效区域 70%~80%
图例：
+ 分类≤4 项：图例置于图表右侧
+ 分类＞4 项：图例置于图表底部，横向排列
+ 禁止饼图分类数量过多，建议最多 8 个分类，多余项合并为「其他」

**图形样式**
+ 区块间隙：相邻区块保留 2px 白色间隙，提升区分度
+ 环形饼图（推荐常用）：内环宽度统一 30%，内环空白无填充

**百分比标注**
+ 占比＜5%：不展示内部文字，仅在 Tooltip 显示
+ 占比≥5%：区块内部标注「名称 + 数值 + 占比 %」

**特殊规则**
+ 单数据项：整块填充主色 Indigo，标注 100%
+ 数据为 0：对应区块隐藏，不在图表占位
+ 内部环保持空白，不放置任何文本。
+ 所有类别名称、数值、百分比仅通过右侧图例和悬浮提示（Tooltip）展示。
+ 图例放置在图表右侧，采用垂直排列，每个图例项的图标与文字保持12px间距，图例与饼图主体保持24px安全间距，避免任何重叠。

### 柱状图专项规范（含纵向 / 横向柱状图）
**坐标轴**

+ X 轴 / Y 轴：轴线颜色 #CDD0D6，刻度线弱化（短细线条）
+ 坐标轴文字：不旋转倾斜（分类名称过长时自动省略，hover 查看全称）
+ 数值轴：根据数据自动均分刻度，刻度值取整数，不保留多余小数

**柱体样式**
+ 基础柱宽：常规容器下柱体宽度 24px，分组柱状图柱体间距 8px
+ 柱体圆角：上下两端统一 4px
+ 单系列：全部使用主色 Indigo
+ 分组 / 堆叠柱状图：按通用配色组依次分配颜色

**数值展示**
+ 柱体顶部可选展示数值，无遮挡、不重叠
+ 负数数据：柱体向下延伸，颜色统一使用危险红 #F53F3F

**分组 / 堆叠扩展规则**
+ 分组柱状图：同组柱体紧密排列，组与组之间保留间距
+ 堆叠柱状图：整体保留统一圆角，分层颜色按配色组执行
+ 确保Y轴标签完整显示，不与边框重叠。
### 表格专项规范
**基础布局与样式**
+ 整体圆角：表格外层容器 8px
+ 边框：采用行分割线模式，仅横向分割线 #CDD0D6，无竖线
+ 内边距：单元格上下 12px、左右 16px

**表头**
+ 背景色：#F2F3F5
+ 文字：14px / 600 / #1D2129，文字左对齐（数字列右对齐）
+ 排序功能：支持排序的列展示升降序图标，激活项使用主色 Indigo

**表格行**
+ 常规行：白底 #FFFFFF，文字 14px / 400 / #4E5969
+ 斑马纹（可选）：偶数行背景 #F7F8FA，弱化区分，不刺眼
+ Hover 行：悬浮行背景 #E9EBEE，整行高亮
+ 选中行：背景使用主色浅底色，文字反白

**单元格内容规则**
+ 文本内容：超长文字自动省略，hover 弹窗展示完整内容
+ 数字内容：统一右对齐，金额 / 大数按业务规则千分位格式化

**状态标签**
+ 成功：绿底标签
+ 警告：橙底标签
+ 异常 / 禁用：红底标签
+ 常规：浅灰标签
+ 标签统一 4px 圆角，字体 12px
1. 功能区域
+ 操作列：固定在表格最右侧，操作按钮优先使用文字按钮样式
+ 分页器：置于表格底部右侧，分页组件配色沿用主色 Indigo
+ 筛选 / 搜索栏：表格顶部，输入框、下拉框遵循全局表单规范
1. 状态适配
+ Loading：表格区域展示加载动画 / 骨架行
+ Empty 无数据：表格居中展示空状态提示，合并整行
+ Error 异常：展示错误文案 + 重试按钮
1. 高级规则
+ 固定列：首列 / 操作列支持左右固定，滚动时层级正常，无错位
+ 合并单元格：仅允许同行 / 同列连续合并，样式与原单元格保持一致
+ 行展开：展开行背景区别于普通行，使用浅灰色底色

## 主页侧边栏导航规范
**触发方式**

+ 父项整行点击触发展开/折叠，右侧搭配箭头图标标识状态（展开向下、收起向右）；各级子项同样支持整行点击

**交互规则**

+ 单选展开：同一层级仅允许一个父项展开子菜单，点击其他同级父项，自动折叠当前已展开的子菜单

+ 点击自身切换：已展开的父项再次点击，折叠对应自身子项

+ 侧边栏整体展开时：所有子项完整显示图标+文字，统一缩进排版、层级间距一致

+ 过渡动画：子项高度、透明度 180ms 平滑过渡，页面无布局抖动

+ 子项基础交互：子项为可点击路由项，点击子项触发页面跳转；点击后仅更新选中状态，不主动折叠所属父菜单

+ 多级子菜单联动：二级子项若包含三级子菜单，遵循和一级父菜单完全一致的展开/折叠、同级互斥规则；同层级所有菜单统一互斥逻辑

+ 跨层级联动：展开下级子菜单时，上级父菜单保持展开状态；折叠上级父菜单，其内部所有下级子菜单同步全部折叠

+ 同级子项行为：同级子菜单之间互不影响，仅同层级父菜单执行单选互斥；子项之间点击切换仅变更选中态，不触发菜单折叠

+ 路由选中联动：当前路由匹配任意子项/父项时，对应菜单自动保持展开状态，对应菜单项高亮选中
